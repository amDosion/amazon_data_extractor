## ✨ 核心功能

### 🎯 主要特性
1. **侧边栏设计** - 固定在浏览器右侧，持久显示
2. **三个API数据源** - Targets、Details V3、Attributes V2
3. **3-Sheet Excel导出** - 自动生成包含3个sheet的报表
4. **60+可提取字段** - 从核心15字段到完整60+字段
5. **批量处理** - 支持一次提取多个ASIN
6. **实时进度** - 进度条、API状态、操作日志
7. **历史记录** - 自动保存最近100条提取记录
8. **配置管理** - 6种预设方案 + 自定义配置
9. **登录检查** - 自动验证Seller Central登录状态
10. **错误处理** - 完善的错误提示和重试机制

### 🚀 技术亮点
- ✅ **Manifest V3** - 使用最新的Chrome扩展规范
- ✅ **Side Panel API** - 侧边栏而非传统popup
- ✅ **Service Worker** - 现代化的后台脚本
- ✅ **并行API调用** - Promise.all优化性能
- ✅ **SheetJS集成** - 强大的Excel生成能力
- ✅ **CSP配置** - 安全的内容安全策略
- ✅ **Chrome Storage** - 持久化配置和历史
- ✅ **响应式设计** - 适应不同宽度的侧边栏

## 🎓 与网页版的区别

| 特性 | 网页版 | 侧边栏扩展 |
|------|--------|-----------|
| 位置 | 独立标签页 | 浏览器侧边栏 |
| 持久性 | 关闭即消失 | 始终可用 |
| 多任务 | 需切换标签 | 边浏览边提取 |
| 权限 | 需在SC页面打开 | 任何页面打开 |
| ASIN识别 | 手动输入 | 自动识别当前页 |
| 历史记录 | 无 | 持久保存 |
| 配置管理 | 无 | 持久保存 |
| 用户体验 | 一次性工具 | 专业工作流 |

**结论**: 侧边栏扩展更适合日常运营使用，网页版适合临时需求。

## 📊 数据提取能力

### 三个数据源
| API端点 | 数据类型 | 典型字段数 |
|---------|----------|-----------|
| Targets | 广告/变体 | 10-15 |

4. **Listing优化** - SEO和内容优化
5. **竞品分析** - 竞争监控
6. **FBA计算** - 物流成本

### 字段优先级
- 🔴 **核心必需** (15个) - 价格、BSR、评论、卖家数...
- 🟡 **重要推荐** (20个) - 浏览量、材质、包装尺寸...
- 🟢 **可选辅助** (25+个) - 保养说明、风格、图案...

## 🚧 已知限制

### 技术限制
1. **仅支持Chrome/Edge** - 不支持Firefox/Safari
2. **需要登录** - 必须有Seller Central账号
3. **API限流** - 批量处理建议间隔1-2秒
4. **CDN依赖** - SheetJS需要网络加载

### 功能限制
1. **只读数据** - 不能修改产品信息
2. **单站点** - 当前仅支持美国站
3. **历史上限** - 最多保存100条记录
4. **无离线模式** - 需要网络连接
